#JB Shortener
A WordPress plugin for modifying the standard shorturl behavior. Adds support for Twitter Tools as well.

#Short Domain
Files are provided for running the short domain. You'll install these on the  The only change that needs to be made is to update index.php with the real domain that it will forward to.

#Setup

1. Buy shortened domain
2. Install plugin
3. Set shorturl in WordPress->Settings->General
4. Set `$domain` variable to real domain in index.php
5. Upload short.domain files to live site
